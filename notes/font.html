Fontes

Fontes podem ter diversas propriedades nas CSS alguma delas incluem:

font-size: 16px; / font-size: 1em;

  • por recomendação do grupo W3C deve-se usar somente 0px e 0em (valores juntos e sem espaço)

  • Geralmente 16px (tamanho padrão) equivale a 1em

  • Nomes de familias de fontes que contem mais de uma palavra separada com um espaço deve-se estar entre aspas

font-wight: #;

  • font-weight: lighter; -> peso mais leve (fino)
  • font-weight: normal; -> peso padrão da fonte (16px, 1em)
  • font-weight: bold; -> peso mais escuro (negrito)
  • font-weight: bolder; -> peso ainda mais escuro (supernegrito)
  • font-weight: 100 ~ 900; -> variação de peso com base numerica indo de 100 (menor peso) até 900 (maior peso)

font-style: #;

  • font-style: italic; => fonte italica
  • font-style: normal; => fonte padrão

text-decoration: #;

  • text-decoration: underline; => decoração do texto

color: #;

  • color: blue; => indica a cor da fonte, nesse caso a cor azul está sendo usada

shorthand font: #;

shorthand significa algo como "uma mãozinha", para facilitar a vida de desenvolvedores de sites o html permite usar varias propriedades font em uma mesma linha. segue abaixo um exemplo

  • font: italic normal 16px monospace

onde a ordem dos parametros são: font: [style] [weight] [size] [family];

devem ser posicionados nessa ordem para que funcione corretamente.
